cuckoo_monitor(pid=-1, h_thread=-1, suspended=False, dll_path=None)

source code 

Invokes injection and resume of the specified process.

Parameters:
  • pid - PID of the process to monitor
  • h_thread - handle of the thread of the process to monitor
  • suspended - boolean value enabling or disabling the resume of the specified process from suspended mode
  • dll_path - path to the DLL to inject, if none is specified it will use the default DLL
